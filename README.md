
# Magi Beta



## Run Locally

Clone the project

```bash
  https://github.com/YounessHassoune/magi-beta
```

Go to the project directory

```bash
  cd magi-beta
```

Run this script in the root folder /magi-beta

```bash
  devspace.exe run-pipeline setup --force-deploy
```

or 
```bash
  devspace run-pipeline setup --force-deploy
```
